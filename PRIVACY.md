# Privacy Policy

**Last Updated: February 2026**

## 1. Data Collection
We collect the following personal information (PII) to provide secure cloud services:
- **Email Address**: For account identity and password recovery.
- **IP Address**: Recorded in security audit logs to prevent unauthorized access and bruce-force attacks.
- **Usage Data**: Metadata about document uploads (file size, type, timestamp).

## 2. Data Protection
- **Encryption**: All document contents are encrypted using AES-256 equivalent cryptography (Fernet).
- **MFA**: We recommend enabling Multifactor Authentication to protect your account.
- **Storage**: Data is stored in a secure local database environment.

## 3. Data Retention
Security logs are retained for 90 days for audit purposes. Documents are stored until explicitly deleted by the user.

## 4. User Rights (GDPR/CCPA)
Users have the right to request a copy of their data or request account deletion. Please contact the administrator for these requests.

## 5. Third Parties
We do not share your personal data with any third-party entities.
